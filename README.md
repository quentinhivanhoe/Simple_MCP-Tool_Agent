# Simple_MCP-Tool_Agent
The goal of this repository is to learn how to use MCP protocol, Connect an AI agent and use it. This project is for beginner like whoe wants to know more about AI.

## Install
If you're using Ubuntu
```bash
$ sudo apt-get update
$ sudo apt-get install python3.6
```
If you're using Fedora
```bash
$ sudo dnf install python3
```

## Setup
To setup the project launch the `setup.sh` file, it will run a virtual environment and install requirements.
